## ğŸš€ Deployment to GitHub Pages

### Quick Setup

1. **Initialize Git Repository** (if not already done):
```bash
git init
git add .
git commit -m "Initial commit: Portfolio website"
```

2. **Create GitHub Repository**:
   - Go to [GitHub](https://github.com/new)
   - Create a new repository named `GouthamVicky` (or any name you prefer)
   - Don't initialize with README (we already have one)

3. **Push to GitHub**:
```bash
git remote add origin https://github.com/YOUR_USERNAME/GouthamVicky.git
git branch -M main
git push -u origin main
```

4. **Enable GitHub Pages**:
   - Go to your repository on GitHub
   - Click on "Settings" tab
   - Scroll down to "Pages" section in the left sidebar
   - Under "Source", select "main" branch
   - Click "Save"

5. **Access Your Site**:
   - Your site will be live at: `https://YOUR_USERNAME.github.io/GouthamVicky/`
   - It may take a few minutes for the site to go live

### Updating Your Portfolio

To update your portfolio after making changes:

```bash
git add .
git commit -m "Update portfolio content"
git push origin main
```

GitHub Pages will automatically rebuild and deploy your site within a few minutes.

## ğŸ“ Customization Guide

### Personal Information

1. Open `index.html`
2. Update the following sections:
   - Name and title in hero section
   - About section content
   - Experience and education timeline
   - Projects information
   - Contact links (LinkedIn, email, GitHub)

### Colors and Theme

1. Open `styles.css`
2. Modify the CSS custom properties in `:root`:
   - `--accent-cyan`: Primary accent color
   - `--accent-green`: Status and success color
   - `--accent-purple`: Secondary accent color
   - `--accent-yellow`: Highlight color

Replace `GouthamVigneshCV.pdf` with your own PDF resume.
